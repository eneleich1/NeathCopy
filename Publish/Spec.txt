
Hola y bendiciones.
Hoy les traigo una aplicación para si gustan la prueben, es gratis. Estaremos contentos de cualquier feedback para poder seguir mejorándola.

Nombre: NeathCopy

Función:
Reemplazar la función de copia de Windows

Requerimientos:
Windows 7 o superior
NetFramework 4.5.2 o superior
VC_redist2017-2019

Enlaces de descarga:
Para sistemas operativos de 64bit: https://eneleich.com/wp-content/uploads/2020/09/NeathCopy_x64.zip
Para sistemas operativos de 32bit: https://eneleich.com/wp-content/uploads/2020/09/NeathCopy_x86.zip

Características:
-Salvar, Cargar lista de archivos a copiar
-En la lista de copia permite mover archivos hacia arriba, abajo, eliminar.
-Permite cambiar el buffer de lectura
-Permite cambiar como se adicionan archivos a las copias existentes: mismo destino, mismo volumen...
-Permite reproducir sonido al cancelar, terminar o adicionar archivos
-Permite tener todas las ventanas en una sola, o por separado
-Permite cuando ocurre un error cerrar o mantenerse abierto
-Permite poner en cola las copias o comenzar automáticamente
-Idiomas disponibles: inglés, español, chino, Frances
-Temas: permite modificar por separado la forma de la ventana externa, la interfaz de copia, los colores y la fuente.
        Incluye Vista previa de los cambios.
-Permite modificar el tiempo de actualización de la información de copia.
-Permite Pausar, Resumir y Cancelar la copia
-Detalles de la copia
	. Nombre de archivo actual, fuente y destino
	. Tiempo restante y transcurrido
	. Cantidad de archivos copiados y restantes
	. Porciento de copia total y del archivo actual
	. Porciento de velocidad de copia
	. Dispositivo destino
-Shell Extension x64 y x86
-Permite ser llamado desde la línea de comandos de Windows (CMD)
	[Para Múltiples archivos, operaciones: copy o move]
	copy "*fichero que contiene los archivos y directorios a copiar" "destino". Ejemplo: copy "*D:\FileList.dat" "D:		\musicales"
	
	[Para un solo archivo]
	copy "G:\Peliculas\The Matrix.mp4" "D:\Peliculas\Favoritas"

Name: NeathCopy

Function:
Replace Windows copy function

Requirements:
Windows 7 or higher
NetFramework 4.5.2 or higher
VC_redist2017-2019

Download links:
For 64bit operating systems: https://eneleich.com/wp-content/uploads/2020/09/NeathCopy_x64.zip
For 32bit operating systems: https://eneleich.com/wp-content/uploads/2020/09/NeathCopy_x86.zip

Features:
-Save, Load list of files to copy
-In the copy list allows to move files up, down, delete.
-Allows to change the read buffer
-Allows changing how files are added to existing copies: same destination, same volume ...
-Lets play sound when canceling, ending or adding files
-Allows you to have all the windows in one, or separately
-Allows when an error occurs to close or stay open
-Allows to queue the copies or start automatically
-Available languages: English, Spanish, Chinese, French
-Themes: allows you to separately modify the shape of the external window, the copy interface, the colors and the font.
        Includes Preview of changes.
-Allows you to modify the update time of the copy information.
-Allows Pause, Resume and Cancel the copy
-Copy details
. Current file name, source and destination
. Time remaining and elapsed
. Number of files copied and remaining
. Percent of total and current file copy
. Percent copy speed
. Target device
-Shell Extension x64 and x86
-Allows to be called from the Windows command line (CMD)
	[For Multiple files, operations: copy or move]
	copy "* file containing the files and directories to copy" "destination". 
	Example: copy "* D:\FileList.dat" "D:\musicals"
	
	[For a single file]
	copy "G:\Peliculas\The Matrix.mp4" "D:\Peliculas\Favorites"
